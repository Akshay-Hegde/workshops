Spatial Database Tips and Tricks

These are the example applications for the workshop.

  1. Map with WMS. OpenLayers with WMS overlay from GeoServer
  2. Click and query. Click the map to run a spatial database query
  3. Click and analyze. Click the map to run a spatial analysis of the surrounding neighborhood.
  4. Click and join. Click the map to perform a spatial analysis based on multiple layers joined together spatially.
  5. Click and union geometry. Pass back vector features from the database, and optionally run a spatial union on them in the database.
  6. Run any SQL query and see the results on the map. Pass back vector features from the database, running any query to retrieve them.
  7. Click and walk a network. Click on a storm drain and the system walks downstream from node-to-node.

See the workshop for more information.
